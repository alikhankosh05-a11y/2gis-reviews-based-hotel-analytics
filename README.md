# Hotel Kazakhstan (Almaty) - 2GIS Reviews Analytics

A powerful **Streamlit dashboard** for analyzing customer reviews of the iconic **Hotel Kazakhstan** in Almaty from 2GIS.

---

## Features

- **Interactive Filters**: Date range, rating range, and text search
- **Key Metrics**: Total reviews, average rating, sentiment breakdown
- **Visualizations**:
  - Rating distribution histogram
  - Sentiment pie chart
  - Monthly trends (reviews + average rating)
  - Word cloud from review texts
  - Top frequent words
- **Sample Reviews** with sentiment labels
- Fully responsive and clean UI
